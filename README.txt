ATS Prototype - PHASE 1:-
Phase 1 contains the basic template and a simple approach to test & show how the ATS would look like externally.
The current prototype has flaws and bugs and is a representation of what was planned on paper.

The features added in Phase 1 are:
	- Login Page
	- Dashboard View (An overview on Total Jobs, Total Applications, Uploaded Resumes)
	- Adding new candidates
	- Job Titles (Creating all Job Titles)
	- Candidates List (An Overview of all applications)
	- Individual Candidate applications
	- Resume Upload page
	- Admin Control Panel (Access to all data from behind the screens)
	- Settings (Edit Display Name, Website Theme, etc)
	- Logout button

Bugs & Modifications that are pending:
	- Skills section needs to be fixed
	- Search Bar needs to be added
	- Redirection on different page to upload resume needs to be fixed
	- Many more features pending...
